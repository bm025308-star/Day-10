# Day 10 – AI-Powered Portfolio Website Development

## 60 Days AI Challenge by AB Talks

### Topic

Building a Personal Portfolio Website Using Claude AI

---

## Objective

To learn how AI can be used to generate a professional portfolio website from personal information, resume details, and project experience with minimal manual coding.

---

## Tasks Completed

### 1. Studied the Learning Resources

Reviewed the provided materials to understand portfolio website structure, personal branding, and AI-assisted web development.

### 2. Watched the Solution Video

Learned the complete workflow for generating a portfolio website using Claude AI.

### 3. Configured Claude

* Opened Claude AI
* Set the effort level to Low
* Started a new conversation

### 4. Used the Portfolio Website Prompt

Provided:

* Personal information
* Educational background
* Skills
* Projects
* Career interests
* Professional achievements

### 5. Enhanced the Portfolio

Optionally included:

* Resume details
* Profile photograph
* Additional project information

### 6. Generated the Portfolio Website

Claude generated a complete portfolio website including:

* Hero Section
* About Me
* Education
* Skills
* Projects
* Certifications
* Contact Information
* Professional Design Layout

### 7. Generated Downloadable HTML

* Created the complete HTML file
* Saved the generated portfolio

### 8. Tested the Website

Verified:

* Layout responsiveness
* Navigation functionality
* Content accuracy
* User experience

### 9. Deployment (Optional)

Explored deployment options using:

* Vercel
* Netlify

---

## Key Learnings

1. AI can generate complete portfolio websites from simple prompts.
2. Personal branding becomes easier with AI-assisted design.
3. Structured information leads to better website quality.
4. AI significantly reduces website development time.
5. Portfolio websites help showcase skills, projects, and achievements professionally.
